Version: 1.8X 1.0    Release: 14th October 2014   [Initial Release]
==============================================================================================================================================================================================================================
-> IPv6 networking has been added. Yes you can now connect to a server with such an address: [2001:41d0:8:1b2c::1]:28961 [New]
-> You can display an external console. [New]
-> Updateoption from Main-Menu. [New]
-> Sending to server current version in connect packet. [New]
-> Added automatic centering of the window if game runs in windowed mode without a border. [New]
-> Scoreboard show now the currently active server map. [New]

-> Linking .dm_1 files with CoD4. Demos can be played now with a doubleclick on this filetype. [New]
-> Implementing the protocol cod4. For example cod4://server.ugiclan.com:28960 is now a valid url for connecting. [New]
-> Build version string which is seen atm. in lower right corner will change now in order to prevent more confusion about which version is really active. [New]
-> Fixed protocol version mismatch. [Fix]
-> Fix a bug with +sysconsole which crashed the server. [Fix]
-> Doesn't raise an error in the event an invalid sampler get used. It will just print to console instead. [Fix]
-> Fix for not properly stored CDKEYs. [Fix]
-> Fix for improper www-download request. [Fix]
-> Fix for autoupdater. [Fix]
-> Fix for the ALT-Key bug. [Fix]
-> Fix for selecting playerprofile twice. [Fix]
-> Hide servers which promoting more than 64 players/slots. [Fix]
-> User Stats won't get encoded anymore. [Fix]
-> Modded servers are now in serverbrowser displayed again properly. [Fix]

-> Userprofiles getting now stored into the AppData\Local\CallofDuty4MW\players directory on Windows7. Prior usersettings/stats are now handled as read only and won't get destroyed. [Fix]

-> Playing demos located into main/demos load now like they are supposed to work (no mod needed). [Fix]
